# SidePet
SidePetは、自分をサポートしてくれるペットを自由に作成し飼うことができます。
また、自分とペット、ペット同士等の交流ができるサービスです。

## Installation
```
git clone git@github.com:momenneko/pet.git
```

## Usage
```
node app.js
```
